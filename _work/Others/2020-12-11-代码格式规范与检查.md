## Prettier 常用配置

```javascript
module.exports = {
  bracketSpacing: true,
  jsxBracketSameLine: false,
  singleQuote: true,
  trailingComma: "all",
  useTabs: true,
  tabWidth: 2,
  arrowParens: "avoid",
};
```

## Eslint 配置

```javascript
module.exports = {
  root: true,
  extends: ["@react-native-community", "plugin:prettier/recommended"],
  rules: {
    "react-native/no-inline-styles": "off",
    "no-console": "warn",
    "no-invalid-regexp": "error",
    "no-empty-pattern": "error",
    "no-var": "warn",
  },
};
```

## husky & lint-staged

- 安装

  `npm i -D husky lint-staged` or

  `yarn add husky lint-staged -dev`

- 配置 `packahe.json`

  ```json
  "husky": {
  		"hooks": {
  			"pre-commit": "lint-staged"
  		}
  	},
  	"lint-staged": {
  		"*.{js,json}": [
  			"prettier --write", // prettier格式化
  			"eslint -c",
  			"git add"
  		],
  	},
  ```
